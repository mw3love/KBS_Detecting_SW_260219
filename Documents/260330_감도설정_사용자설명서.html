개요

감도설정이란?

프로그램은 200ms(0.2초)마다 영상 프레임을 분석하여 이상 여부를 판단합니다.
감도설정은 이 판단 기준을 조절하는 파라미터 모음입니다.

⬛

블랙 감지

화면이 검게 변했을 때 감지.
픽셀 밝기 기반.

🖼️

스틸 감지

화면이 정지(스틸)됐을 때 감지.
프레임 간 변화량 기반.

🎚️

오디오 레벨미터

화면에 표시된 레벨미터 색상을 HSV로 분석.
색이 사라지면 무음으로 판단.

🔊

임베디드 오디오

수신된 오디오 신호의 dB를 직접 측정.
무음 지속 시 알림.

💡 공통 원칙: 모든 감지는 설정된 시간만큼 이상 상태가 지속되어야 알림이 발생합니다. 일시적인 노이즈나 단일 프레임 이상으로는 알림이 울리지 않습니다.
공통 개념

감지 공통 동작 원리

감지영역 (ROI)

전체 화면 중 분석할 영역을 지정합니다.
영역이 작을수록 CPU 부하가 낮고, 여러 개 지정 가능합니다.
비디오 ROI는 블랙/스틸 감지에, 오디오 ROI는 레벨미터 감지에 사용됩니다.

블랙 감지가 안 될 때 — 조절 가이드

기본값으로 감지가 안 되는 경우, 아래 순서대로 확인하고 조절하세요.

1️⃣

감지영역(ROI) 위치 확인

ROI가 화면의 블랙이 발생하는 부분을 제대로 덮고 있는지 확인합니다.
자막이나 로고가 포함된 영역을 피해 순수 영상 영역에 ROI를 설정하세요.

2️⃣

어두운 픽셀 비율 임계값 낮추기 (98% → 85~90%)

화면이 블랙이어도 자막·로고·오버레이 때문에 일부 픽셀이 밝게 남아 있는 경우입니다.
어두운 픽셀 비율을 98% → 85%로 낮춰보세요.
⚠️ 너무 낮추면 정상 영상도 블랙으로 오감지될 수 있습니다.

3️⃣

밝기 임계값 올리기 (5 → 10~15)

인코더 또는 전송 과정에서 완전 검정이 아닌 약간 밝은(값 5~10) 픽셀로 변환되는 경우입니다.
밝기 임계값을 5 → 10으로 올리면 약간 밝은 픽셀도 어둡다고 인식합니다.

4️⃣

모션 억제 비율 확인 (스크롤 자막 오판 케이스)

블랙인데 스크롤 자막이 있어서 감지가 취소되는 경우, 모션 억제 비율을 0.0으로 설정하거나
자막이 없는 영역만으로 ROI를 재설정합니다.

5️⃣

알림 발생 시간 조정 (20초 → 10초)

블랙이 짧게 발생(예: 10초)하는 경우 기본값(20초)에서 알림이 안 울립니다.
알림 발생 시간을 짧게(예: 10초) 줄여 더 빠르게 감지하도록 합니다.

권장 조절 순서: ROI 위치 확인 → 어두운 픽셀 비율 낮추기 → 밝기 임계값 올리기 → 모션 억제 조정 → 알림 시간 조정

스틸 감지

스틸 감지가 안 될 때 — 조절 가이드

케이스 A: 스틸인데 타이머가 계속 리셋될 때

화면은 정지해 있는데 인코더 압축 노이즈 때문에 1~2프레임씩 "움직임 있음"으로 오판정되는 경우입니다.

🔧

픽셀 변화 기준값 올리기 (4 → 8~12)

노이즈 크기보다 기준값을 높여 실제 움직임만 인식하게 합니다.

🔧

타이머 리셋 프레임 수 올리기 (3 → 5~7)

1프레임의 노이즈로 타이머가 리셋되지 않게 합니다.

케이스 B: 화면이 진짜 변하는데 스틸로 오감지될 때

자막 티커나 시계 등 작은 움직임만 있을 때 블록 임계값이 너무 높아 움직임을 인식 못 하는 경우입니다.

🔧

블록 움직임 임계값 낮추기 (10% → 3~5%)

작은 움직임도 움직임으로 판정하게 합니다.

🔧

픽셀 변화 기준값 낮추기 (4 → 2)

미세한 픽셀 변화도 변화로 인식하게 합니다.

HSV 색상 범위란?

레벨미터의 초록색 바를 픽셀 단위로 검출합니다.
HSV(색상·채도·명도) 색상 공간에서 범위를 지정합니다.

H (색상, Hue): 40 ~ 95
0 (빨강)60 (노랑)120 (초록)180 (청록)
↑ 이 범위(노란-초록-청록 계열)를 감지
S (채도): 80 ~ 255 — 탁한 색 제외, 선명한 색만
V (명도): 60 ~ 255 — 너무 어두운 색 제외
레벨미터 감지 원리 요약
소리 있음 → 초록 바 표시 → ratio 높음 → 정상
소리 없음 → 바 사라짐 → ratio 낮음 → 이상

오디오 레벨미터

오디오 레벨미터 감지가 안 될 때 — 조절 가이드

레벨미터가 화면에 보이는데도 감지가 안 되는 경우, 주로 HSV 색상 범위 불일치가 원인입니다.

1️⃣

오디오 ROI 위치 확인

ROI가 레벨미터 바의 초록 부분을 정확히 포함하는지 확인합니다.
ROI가 레벨미터 테두리나 배경 영역에만 걸쳐 있으면 초록 픽셀이 감지되지 않습니다.

2️⃣

H(색상) 범위 확장 — 레벨미터 색이 다른 경우

레벨미터가 노란색·청록색 등 다른 색상인 경우입니다.
H_min을 20~30으로 낮추거나, H_max를 120~130으로 올려 범위를 넓혀보세요.

3️⃣

S(채도) 최솟값 낮추기 — 색이 흐릿한 경우

레벨미터가 연한 색(채도 낮음)인 경우입니다.
S_min을 80 → 40~60으로 낮춰 흐릿한 색도 감지하도록 합니다.

4️⃣

V(명도) 최솟값 낮추기 — 화면이 어두운 경우

레벨미터 배경이 어두워서 바 색상도 어둡게 보이는 경우입니다.
V_min을 60 → 30~40으로 낮춰보세요.

5️⃣

감지 픽셀 비율 낮추기 (5% → 2~3%)

레벨미터 바가 ROI 대비 차지하는 면적이 작은 경우입니다.
감지 픽셀 비율을 2~3%로 낮추면 적은 초록 픽셀로도 활성으로 판정합니다.

HSV 조절 팁

변경 후 로그 창에서 실시간으로 ratio 값을 확인할 수 있습니다. 소리가 있을 때 ratio가 5% 이상이고, 소리 없을 때 5% 미만이면 설정이 올바릅니다.

dB 값 이해하기

dB(데시벨)는 소리 크기를 나타냅니다. 0에 가까울수록 크고, 음수가 클수록 작습니다.
0 dB최대 음량
-12 dB일반 방송 수준
-30 dB아주 작은 소리
-50 dB← 기본 무음 임계값
-60 dB거의 완전한 무음
임계값을 -30 dB로 올리면 작은 소리도 무음으로 판단
임계값을 -60 dB로 낮추면 거의 완전한 무음만 감지

→ 블록 임계값 낮추기 10% → 5%
