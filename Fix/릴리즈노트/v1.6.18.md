# v1.6.18 릴리즈 노트

**릴리즈 날짜**: 2026-04-10

## 변경사항 요약

### 기능 개선
- 예약 자동 재시작을 하루 2회까지 설정 가능하도록 확장
  - 설정 > 시스템 탭에 "재시작 시각 2" 체크박스+시각 위젯 추가
  - 체크 해제 시 시각 2 비활성(빈 문자열 저장), 체크 시 활성
  - 기존 "시각 1"은 동작 방식 그대로 유지 (하위 호환)

### 내부 구조 변경
- `_restart_done_date: str` → `_restart_done_slots: set` 교체
  - 슬롯 키: `"YYYY-MM-DD_HH:MM"` 형태로 각 시각을 독립 추적
  - 시각 1 재시작 후 시각 2가 차단되지 않음
  - `--restarted HH:MM` 인자도 해당 슬롯만 done 처리 (나머지 시각 유효)
- config에 `scheduled_restart_time_2` 키 추가 (기본값 `""`)

## 커밋 이력

이번 버전은 v1.6.17 이후 직접 작업한 변경사항으로 구성됩니다.
