# v1.6.20 릴리즈 노트

**릴리즈 날짜**: 2026-04-12

## 변경사항 요약

### 버그 수정
- DIAG-AUDIO 섹션에서 `_embedded_alert_start`가 비-float 타입일 경우 발생하던 TypeError 재발 차단
  - 로컬 캡처(v1.6.17) 이후에도 TypeError가 재발하던 문제 — 타입 검증 + None 강제 방어로 완전 차단
  - 재발 시 실제 타입/값을 로그에 기록하여 근본 원인 추적 가능

### 개선
- Watchdog(Health Check) 감지 루프 중단 시 로그 상세 정보 추가
  - `detect_timer`, `latest_frame`, `py_threads` 추가 — freeze 유형 즉시 분류 가능
- Watchdog 감지 루프 중단 감지 시 텔레그램 즉시 발송
  - 운영 중 freeze 발생 시 운영자가 즉각 인지하여 로그 수집 가능
